# FairySplit — How It Works

FairySplit helps groups split USDC expenses and settle onchain.

## 1) Create a split
- Open `Create split`.
- Enter split name, optional description, and optional reminder note.
- Add participants (wallet addresses or supported names).
- Set the per-person amount in USDC.

## 2) Share and track
- Participants appear in your bill view and requests flow.
- Each participant has a clear paid/unpaid status.
- Activity logs include chain labels and transaction links for verification.

## 3) Settle payments
- **Direct payment:** Standard USDC transfer and onchain confirmation.
- **Confidential flow:** Use confidential wallet actions when privacy is needed.
- If confidential funds show as pending, use **Claim pending cUSDC**.

## 4) Stay coordinated
- Send reminders and keep split status updated.
- Use profile and friends features for better identity and contact handling.
- Review all completed actions in Activity.

## Supported chains (current)
- Base Sepolia
- ARC Testnet

FairySplit is built for practical, verifiable group payments with optional privacy.
